# samp-chatlog
Mod (.asi) to avoid sudden chatlog losses!

## Example
The new location for chatlog backups will be here: `GTA San Andreas User Files` -> `ChatLogs` (Folder) -> `chatlog YEAR-MOUNTH-DAY.txt` (Ex: chatlog 2025-12-23.txt)
New Format:
```md
[2025-12-23 21:25:55] {FFFFFF}SA-MP {B9C9BF}0.3.7-R5 {FFFFFF}Started

[2025-12-23 21:25:58] Connecting to xxx.xx.xxx.xx:7777...

[2025-12-23 21:26:00] Connected. Joining the game...

[2025-12-23 21:26:05] Connected to {B9C9BF}Server


```
